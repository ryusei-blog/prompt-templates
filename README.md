AI用プロンプトを再利用可能なYAMLテンプレート集です。  
ブログ記事を書くすべてのライター・マーケター向けに設計。  
コピペするだけで構成作成から引用元探しまで時短できるプロンプトを色々と取り揃えていきます！

## 📑 テンプレート一覧

| ファイル | 役割 / 使いどころ | プロンプト内容 |
|---------|-----------------|------------------|
| `primary_source_finder.yml` | 渡されたMarkdown 記事から**引用・統計などが書かれた箇所を抽出　→　一次情報 URL を検索**し、Markdown テーブルで返す。ChatGPT o3に対して使うのがおすすめ。 | [コピー用ページを開く](https://ryusei-blog.github.io/prompt-templates/) |
